# Lost Signal — Game Design

## Core Idea
A point-and-click adventure about a researcher named Nilo who investigates a series of mysterious signals originating from beneath the Taren surface.

## Player Role
Players take control of Nilo, navigating through cinematic scenes, collecting items, and solving environmental puzzles to uncover the truth of the "Lost Signal".

## Core Mechanics
- **Point-and-Click Navigation**: Intuitive interaction with the world.
- **Inventory System**: Collecting and combining/using items.
- **Environmental Puzzles**: Solving problems using logic and the tools found.
- **Bilingual Narrative**: Storytelling through dialogues in English and Swedish.

## The First Scene: The Dock
Nilo arrives at an old, abandoned dock. This serves as the tutorial area.
- Goal: Restore power to the main signal station door.
- Puzzle: Find the missing fuse and install it.

## Future Chapters
1. **The Machine Room**: Decoding the first signal fragment.
2. **The Descent**: Entering the tunnels beneath Sector 7.
3. **The Core**: Meeting the source of the signal.
