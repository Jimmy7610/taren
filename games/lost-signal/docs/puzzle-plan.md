# Lost Signal — Puzzle Plan

## Current Build (001)
### Puzzle 1: Power the Station
- **Logic**: The door to the station is powerless.
- **Requirement**: Find the `Rusty Fuse` and use it on the `Rusty Fuse Box`.
- **Location**: The fuse is found by investigating the "Old Sign" (simulating a search in a nearby crate).
- **Solution**: Select the fuse from the inventory and click the fuse box.

## Future Builds
### Puzzle 2: The Door Code
- **Logic**: The door is powered but requires a 4-digit code.
- **Clue**: The code is hidden in the reflections of the water at the dock.
- **Solution**: Enter the code into the keypad hotspot.

### Puzzle 3: The Frequency Match
- **Logic**: Inside the machine room, a terminal is receiving static.
- **Requirement**: Match the frequency of the receiver to the incoming signal.
- **Mechanic**: Mini-game involving slider adjustments.
